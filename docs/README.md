# Documentation Index

This folder contains submission-ready documentation for the PrenurLab full-stack assessment.

## Files
- `requirement-compliance-matrix.md`: exact requirement-to-implementation mapping with PASS status.
- `step-by-step-build-log.md`: implementation timeline and execution phases.
- `design-decisions.md`: architecture choices and tradeoff rationale.
- `code-walkthrough.md`: module-level explanation of backend and frontend code.
- `prompt-log.md`: AI tool usage and prompt history summary.
- `postman/PrenurLabTask.postman_collection.json`: Postman collection for API testing.
